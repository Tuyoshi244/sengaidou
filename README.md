# フリースペース泉崖堂予約システム

愛知県岡崎市康生通りにある泉崖堂の利用予約システムのフロントページ。

Code for AICHIで委託作成。

# 作成した経緯

利用している市民の方たちからの要請で作成。

また、空き店舗である泉崖堂の利用促進を図る。

# システム詳細

レスポンシブの一般的なHTML、CSS、JavaScriptで構成。
